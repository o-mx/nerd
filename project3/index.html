Introduction

HTML (HyperText Markup Language) is the foundational language used to create web pages and web applications. It structures and organizes content on the web by defining elements such as headings, paragraphs, images, links, and multimedia. Invented by Tim Berners-Lee in 1991, HTML has grown into the backbone of the internet, allowing developers to create everything from simple websites to complex, interactive applications.

To better understand HTML, think of it as the "skeleton" of a webpage. It works hand-in-hand with CSS (Cascading Style Sheets) for design and JavaScript for interactivity, forming the holy trinity of web development.

In order to learn HTML it is essential to have to basic skills like:

  • Basic computer literacy: Understanding how to use a computer, navigate file systems and some knowledge on applications like Ms word, can improve your understanding and the time used to learn.
  • Problem-solving skills: The ability to trouble shoot issues with code and use logical thinking can help to figure out solutions and improve creativity.
  • Time management and self learning: Learning HTML involves a lot of self-paced study and practice, Managing time effectively to balance learning with practical projects or assignments.
  • Attention to Detail: HTML requires precise syntax (e.g., matching opening and closing tags). A small mistake in one tag can cause the entire page to not display correctly, so attention to detail is crucial.
  • Research Skills: Knowing how to use online resources effectively (e.g., Stack Overflow, MDN Web Docs) when you get stuck or need to learn more about certain HTML tags or practices. Using search engines to find tutorials, examples, and troubleshooting guides.

Structure of an HTML Document

Every HTML document adheres to a specific structure, starting with the declaration, which tells the browser to interpret the document as HTML5. The "html" element wraps the entire content, divided into the "head" (for metadata) and "body" (for visible content).

Example: 

 
            <html>
              <head>
                <title>Welcome to HTML</title>
              </head>
              <body>
                <h1>Hello, World!</h1>
                <p>This is a basic HTML structure.</p>
              </body>
            </html>
Basic Elements of HTML

HTML provides core elements to structure and format content. These include headings (<h1> to <h6>), paragraphs (<p>), links (<a>), and lists (<ul> and <ol>). These elements ensure content is organized and user-friendly.

Example:

 <h1>Main Heading</h1>
    <p>This is a paragraph.</p>
    <a href="https://example.com">Visit Example</a>
    <ul>
      <li>Item 1</li>
      <li>Item 2</li>
    </ul>
Attributes in HTML

Attributes in HTML provide additional information about elements and modify their behavior or appearance. Each attribute is placed within the opening tag of an HTML element and is composed of a name and a value, typically written in a key-value pair format. These attributes are essential for defining specific properties such as links, image sources, or styles, enhancing the functionality and customization of web pages.

One of the most commonly used attributes is the href attribute in anchor (<a>) tags, which specifies the destination of a hyperlink. Other popular attributes include src in image (<img>) tags, which defines the image source, and alt, which provides an alternative text for accessibility. Similarly, the class and id attributes allow developers to target elements with CSS or JavaScript, making it easier to apply styles or add interactivity to specific elements.

Example:

 <img src="image.jpg" alt="An example image" width="300">
    <a href="https://example.com" target="_blank">Open Example</a>
